本プラグインパッケージco.honobono.hncorefix.util.japaneseは
うっちぃ様作成のLunaChat(https://github.com/ucchyocean/LunaChat)内にある
パッケージcom.github.ucchyocean.lc.japanize内の3ファイルを使用しています

本プラグインはMaxMindDBを使用します。
Pluginを起動後[ http://dev.maxmind.com/ja/geolite2/ ]よりGeoLite2-City.mmdb.gzをダウンロードし
解凍後plugin/${pom.artifactId}/MaxMindDB内に保存してください。
